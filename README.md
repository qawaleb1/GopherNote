# 📝 GopherNote

Простое веб-приложение для создания и управления заметками с поддержкой Markdown-синтаксиса, построенное на **Blazor** WebAssembly.

## ✨ Возможности

- ➕ Создание, редактирование и удаление заметок
- 💾 Автоматическое сохранение в localStorage в вашем любимом браузере
- 🏷️ Категории/теги для заметок 
- 🌙 Тёмная/светлая тема 
- 📌 Закрепление важных заметок вверху списка

## 🛠️ Технологии

- **Фреймворк:** Blazor .NET 10
- **Тип:** Blazor WebAssembly
- **UI библиотека:** Bootstrap
- **Хранение данных:** 
  - Локально: `localStorage` 

## 📋 Требования

- [.NET SDK](https://dotnet.microsoft.com/download) (версия 8.0 или выше)
- Браузер на основе Chromium
- C# 12 и выше

## 🚀 Установка и запуск

Клонируйте репозиторий командой `git clone https://github.com/Vo1dS/GopherNote.git`, затем перейдите в папку проекта `cd GopherNote`, восстановите все зависимости командой `dotnet restore` и запустите приложение: используйте `dotnet run --project GopherNote`.

## 📁 Структура проекта

В папке GopherNote.Client находятся компоненты Components, страницы Pages, сервисы Services, модели Models и статические файлы wwwroot. В папке GopherNote.Server расположены API контроллеры Controllers, контекст базы данных Data и файл настроек appsettings.json. Общие модели и утилиты вынесены в GopherNote.Shared.

## 💻 Использование

Для создания заметки нажмите кнопку "Новая заметка", введите заголовок и содержание, затем сохраните. Для редактирования кликните по кнопке редактирования. Для удаления нажмите кнопку удалить. Для поиска используйте строку поиска слева страницы — поиск работает по тегам заметок.

## 🔧 Конфигурация

Для смены типа хранения данных откройте файл Program.cs и измените регистрацию сервиса: для localStorage используйте `builder.Services.AddScoped<INoteService, LocalStorageService>()`.
